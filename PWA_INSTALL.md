# PWA(アプリ)インストール手順

## 概要

このオーダーシステムをPWA(Progressive Web App)としてインストールすると、以下のメリットがあります:

✅ URLバーが表示されず、アプリのようなフルスクリーン表示
✅ ホーム画面から直接起動できる
✅ オフラインでも一部機能が使用可能(キャッシュ)

---

## Androidタブレットでのインストール方法

### ホール用画面(index.html)をインストール

1. **ブラウザでアクセス**
   - ChromeまたはEdgeで `https://あなたのGitHubPages URL/index.html` を開く

2. **メニューを開く**
   - 画面右上の「︙」(縦3点メニュー)をタップ

3. **ホーム画面に追加**
   - 「ホーム画面に追加」または「アプリをインストール」をタップ
   - 名前を確認(「オーダー」または好きな名前)
   - 「追加」をタップ

4. **完了**
   - ホーム画面に「オーダー」アイコンが追加されます
   - このアイコンから起動すると、URLバーなしで表示されます

### キッチン用画面(kitchen.html)をインストール

1. **ブラウザでアクセス**
   - ChromeまたはEdgeで `https://あなたのGitHubPages URL/kitchen.html` を開く

2. **メニューを開く**
   - 画面右上の「︙」(縦3点メニュー)をタップ

3. **ホーム画面に追加**
   - 「ホーム画面に追加」または「アプリをインストール」をタップ
   - 名前を確認(「キッチン」または好きな名前)
   - 「追加」をタップ

4. **完了**
   - ホーム画面に「キッチン」アイコンが追加されます

## iPadでのインストール方法

### Safari(iOS/iPadOS)の場合

1. **Safariでアクセス**
   - Safari で `https://あなたのGitHubPages URL/index.html` を開く

2. **共有ボタンをタップ**
   - 画面下部の「共有」ボタン(□に↑マーク)をタップ

3. **ホーム画面に追加**
   - 「ホーム画面に追加」をタップ
   - 名前を確認して「追加」をタップ

4. **完了**
   - ホーム画面にアイコンが追加されます

**注意**: iOSのSafariでは、Service Workerの対応が限定的なため、一部の機能が制限される場合があります。

## 運用イメージ

### 朝の準備(キッチン側)

1. ホーム画面の「キッチン」アイコンをタップして起動
2. 画面上部の **🔔 通知音テスト** ボタンを1回タップ
3. 「ピロン♪」という音が鳴れば準備完了
4. これで、その日はホールから注文が入るたびに自動で通知音が鳴ります

### ホール側

1. ホーム画面の「オーダー」アイコンをタップして起動
2. 通常通り注文を入力
3. 「注」ステータスにすると、キッチン側で自動的に通知音が鳴ります

## トラブルシューティング

### 「ホーム画面に追加」が表示されない

- **原因**: HTTPSでアクセスしていない可能性があります
- **対処**: GitHub Pagesは自動的にHTTPSになりますが、URLが `https://` で始まっていることを確認してください

### インストールしたアイコンが表示されない

- **原因**: アイコン画像が配置されていない可能性があります
- **対処**: `ICON_README.md` を参照して、アイコン画像を作成・配置してください

### 通知音が鳴らない

- **原因**: AudioContextが初期化されていない可能性があります
- **対処**: キッチン画面上部の「🔔 通知音テスト」ボタンを一度タップしてください

### オフラインで動作しない

- **原因**: Service Workerがまだキャッシュを作成していない可能性があります
- **対処**: 一度オンラインでアクセスしてから、オフラインで試してください

---

## アンインストール方法

### Androidの場合

1. ホーム画面のアイコンを長押し
2. 「削除」または「アンインストール」をタップ

### iOSの場合

1. ホーム画面のアイコンを長押し
2. 「Appを削除」をタップ

---

## 補足

- PWAは通常のウェブアプリなので、ブラウザのブックマークやタブからもアクセスできます
- インストールは任意です。通常のブラウザでの使用も可能です
- アイコンや名前は後から変更できません。変更したい場合は一度アンインストールして再インストールしてください
