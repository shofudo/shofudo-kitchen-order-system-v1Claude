# PWA アイコンの準備方法

## 必要なファイル

GitHub リポジトリの`icons`フォルダに、以下の 2 つの PNG 画像を配置してください:

- `icon-192.png` (192x192 ピクセル)
- `icon-512.png` (512x512 ピクセル)

## アイコンのデザイン案

### ホール用(index.html)

- 背景色: #8b7355 (茶色系)
- アイコン: 📋 (クリップボード) または 🍽️ (料理)
- テキスト: 「注」「ORDER」など

### キッチン用(kitchen.html)

- 背景色: #3498db (青系)
- アイコン: 🍳 (フライパン) または 👨‍🍳 (シェフ)
- テキスト: 「厨」「KITCHEN」など

## オンラインツールで簡単に作成する方法

### 方法 1: Canva を使用

1. https://www.canva.com/ にアクセス
2. 「カスタムサイズ」で 512x512 ピクセルを選択
3. 背景色と絵文字を配置
4. PNG でダウンロード
5. 192x192 版は、ダウンロード後にリサイズ

### 方法 2: favicon.io を使用

1. https://favicon.io/favicon-generator/ にアクセス
2. テキストや絵文字を入力
3. 背景色とフォントを設定
4. 生成されたファイルから必要なサイズを使用

### 方法 3: 画像編集ソフト

- Photoshop、GIMP、Paint などで作成
- サイズ: 512x512 ピクセルと 192x192 ピクセル
- 形式: PNG
- 背景: 透過なしの単色がおすすめ

## ファイルの配置

作成したアイコンを以下のように配置してください:

```
your-repo/
├── index.html
├── kitchen.html
├── manifest.json
├── manifest-kitchen.json
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## 注意事項

- アイコンは正方形で作成してください
- ファイル名は正確に `icon-192.png` と `icon-512.png` にしてください
- 背景は透過ではなく、単色がおすすめです(Android では透過部分が白くなることがあります)
- シンプルで視認性の高いデザインがおすすめです

## アイコンがない場合

アイコンファイルがない状態でも PWA として動作しますが、以下のような表示になります:

- ホーム画面のアイコンがデフォルトのブラウザアイコンになる
- または、ウェブサイトのファビコンが使用される

本格運用前に、必ずアイコンを準備することをおすすめします。
